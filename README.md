# PlanMate - 智能 AI 目标规划助手 🚀

PlanMate 是一款基于微信小程序开发的智能目标管理与日程规划工具。它通过集成大语言模型（LLM），将用户的自然语言描述自动转化为结构化的可执行计划，帮助用户更高效地管理生活与工作。

---

## ✨ 核心特性

-   **🤖 AI 智能规划**：支持通过自然语言输入计划，AI 自动提取目标、分类、日期及任务优先级。
-   **🔌 多模型支持**：内置 **通义千问 (Aliyun Qwen)** 与 **Google Gemini** 双引擎，支持多种模型版本（Turbo, Plus, Max, Flash, Pro）切换。
-   **📅 多维目标管理**：支持 **日计划 (Daily)**、**周目标 (Weekly)** 与 **年规划 (Yearly)**，全方位覆盖个人成长。
-   **📊 数据可视化**：自动统计任务完成情况，通过直观的图表展示进度与效率。
-   **🌓 优雅 UI 设计**：采用现代简约风格，提供流畅的交互体验与清晰的角色化图标。

---

## 🛠️ 技术栈

-   **框架**：原生微信小程序 (WXML, WXSS, JS)
-   **AI 集成**：OpenAI 兼容接口 (DashScope / Gemini API)
-   **存储**：微信本地存储 (Sync Storage)
-   **脚本**：Python (用于自动化生成资源)

## 🚀 快速开始

### 1. 克隆项目
```bash
git clone https://github.com/joeeei11/-LLM-.git
```

### 2. 导入项目
打开 **微信开发者工具**，选择「导入项目」，目录指向克隆下来的文件夹。

### 3. 配置 API Key
由于安全原因，API Key 不会包含在代码中。请按照以下步骤配置：
1.  在小程序中进入 **「我的」** 页面。
2.  点击 **「设置 API Key」**。
3.  选择服务商（通义千问或 Gemini）并填入您的 API Key。
4.  **重要提示**：在开发者工具中，点击右侧 **「详情」->「本地设置」**，勾选 **「不校验合法域名」** 以允许请求 AI 接口。
