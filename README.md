# proj-WechatMiniApp-LLM-HealthManager

**接入大语言模型的微信健康管理小程序，提供 AI 健康计划、日历打卡、数据统计等功能。**

## 功能特性

- LLM 智能生成个人健康计划
- 日历式打卡与历史记录
- 健康数据可视化统计图表
- AI 反馈与行为分析
- 用户个人档案与偏好设置

## 快速开始

### 环境要求

- 微信开发者工具 >= 1.06
- 已开通微信云开发或自行部署后端

### 安装步骤

```bash
git clone https://github.com/joeeei11/proj-WechatMiniApp-LLM-HealthManager.git
```

用微信开发者工具打开，在 `app.js` 中配置 LLM API Key 和云开发 envId。

### 基础用法

```
微信开发者工具 → 预览 / 真机调试
```
